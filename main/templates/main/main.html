HTML

HTML – це будівельні блоки сторінки сайту. За допомогою нього задаються структури на сайті та їх мінімальні властивості (що залежать від типу структури). За структури HTML відповідають теги, а за властивості - атрибути.

Синтаксис парних тегів:

<Назва тегу та його атрибути>...</Назва тегу>

Синтаксис одиночних тегів:

<Назва тегу та його атрибути>

Структура HTML

1) Оголошення HTML файлу.

<!DOCTYPE html>

2) Голова файлу де оголошуються назва сторінки та підключаються зовнішні файли (CSS-файл, файли з кодами тощо). Голова HTML задається парним тегом <head>. Приклад:

<head>

<title>Назва сторінки</title> - назва сторінки

<link rel="stylesheet" type="text/css" href="CSS.css"> - підключення CSS

</head>

3) Тіло файлу, де прописується основний контент. Тіло сайту оголошується парним тегом <body>.

Таблиці та блоки

Таблиці та блоки задають "склет" сайту. Вони відрізняються тим, що таблиця вже має структуру за замовчуванням, а блоки створюються незалежно і вимагають тонкої настройки. Таблиці оголошуються парним тегом <table>, а блоки - парним тегом <div>.

Розглянемо синіаксис таблиць

1) Насамперед формуються рядки за допомогою парного тега <tr>

<table>

<tr></tr>

<tr></tr>

<tr></tr>

<tr></tr>

</table>

Кожна пара <tr></tr> задає один рядок

2) Усередині тега <tr></tr> формуємо комірки заголовка та стовпці таблиці.
За заголовні комірки відповідає парний тег <th>.
А за стовпці – парний тег <td>.

Код

<table>

<tr><th>заголовок 1</th><th>заголовок 2</th></tr>

<tr><td>комірка 1.1</td><td>комірка 2.1</td></tr>

<tr><td>комірка 1.2</td><td>комірка 2.2</td></tr>

<tr><td>комірка 1.3</td><td>комірка 2.3</td></tr>

</table>

І маємо
заголовок 1заголовок 2
комірка 1.1комірка 2.1
комірка 1.2комірка 2.2
комірка 1.3комірка 2.3

3) У деяких випадках нам потрібно об'єднати деякі осередки в одну.
Точніше не поєднати, а розтягнути.
За об'єднання рядів відповідає тег <rowspan>.
А за об'єднання стовпців <colspan>;.
Ці теги поміщають усередину тега <td> таким чином:
<td colspan(или rowspan)="к-сть стовпців (або рядів) які ви хочете щоб цей осередок "поглинув" в себе">.

Код

<table>

<tr><th>заголовок 1</th><th>заголовок 2</th></tr>

<tr><td colspan="2">комірка 1.1</td></tr>

<tr><td rowspan="2">комірка 1.2</td><td>комірка 2.2</td></tr>

<tr><td>комірка 2.3</td></tr>

</table>

І маємо

заголовок 1заголовок 2
комірка 1.1
комірка 1.2комірка 2.2
комірка 2.3
(були також додані рамки щоб краще бачити результат)

Зверніть увагу що комірки, які були "поглинені" (тобто 2.1 і 1.3) в коді не прописуються.

Тепер розглянемо атрибути застосовні для тегів <table>, <div>, <tr>,<td> та інших.

1) Атрибути width та height.
Атрибути width та height відповідає за довжину та висоту відповідно.
Вони можуть приймати значення px,em або в %.

<table width="10px" height="10%"></table>

2) Атрибути valign та align.
valign - Вертикальне вирівнювання.
align - Горизонтальне вирівнювання.
Вони можуть набувати значеньtop, right, bottom, left

<tr valign="top" align="left"></tr>

3) Атрибут border (рамка).

<th border="1"></th>

4) Атрибути id та class прив'язують тег з CSS.

Інщі елементи сторінки

1) Заголовки на сторінці викликаються через парні теги <h1>,<h2>...<h6>. Де <h1> - головний заголовок, <h2> - заголовок другого порядку і тд.

<h1>Заголовок 1</h1>

<h2>Заголовок 2</h2>

<h3>Заголовок 3</h3>

<h4>Заголовок 4</h4>

<h5>Заголовок 5</h5>

<h6>Заголовок 6</h6>

Заголовок можно підключити до CSS за допомогою атрибутів id або class.

2) Параграфи викликаються парним тегом <p>

<p>Перший параграф</p>

<p>Другий параграф</p>

Параграф можно підключити до CSS за допомогою атрибутів id або class.

3) Посилання викликаються парним тегом <а>. Адресса посилання задається атрибутом href

<a href="посилання">Натисни мене</a>

Можна підключити до CSS за допомогою атрибутів id або class.

4) Кнопка викликаються парним тегом <button>. Можна підключити до CSS за допомогою атрибутів id або class. Можна підключити до кодового файлу за допомогою атрибута id.

<button>Натисни мене</button>

5) Зображення викликаються тегом <img>. Розташування картинки задається атрибутом src. Для картинки можна підключити тіж самі атрибути, що і для таблиць.

<img src="розташування">

CSS

CSS файл відповідає за стилістику об'єктів. Один CSS файл може бути підключений до багатьох html сторінок, що спрощує стилізацію сайту вцілому.

Синтаксис CSS:

1) Класи об'єктів викликаються символом ".". Наприклад:

.Bord

{border:5px solid;}

Всі об'єкти, що були додані до класу Bord тепер мають неперервну рамку товщиною 1 піксель.

2) Обєкти певного id викликаються символом "#". Наприклад:

#Name1

{background-color: black;

font-family: Calibri;}

Всі об'єкти, що мають id Name1 тепер мають чорний фон та шрифт тексту Calibri.

3) Можна модифікувати конкретний об'єкт:

h2

{font-family:Calibri;}

Всі другорядні заголовки тепер мають шрифт Calibri.
